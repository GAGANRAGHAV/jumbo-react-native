## Get started

1. Clone Repository 

   ```bash
   git clone https://github.com/GAGANRAGHAV/jumbo-react-native.git
   ```

2. Install Dependencies

   ```bash
   npm install
   ```
3. Start the app

   ```bash
   npx expo start
   ```
